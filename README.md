### Para a aplicação Web

- Node 22+
- Faça o clone desse projeto
- Dentro da pasta do projeto, execute ```npm i -f```
- Para subir as aplicações, execute ```npm run start:dev```
- Acesse em: [localhost](//localhost:9000/).

### Para a aplicação Mobile

- Node 22+
- Faça o clone desse projeto
- Dentro da pasta do projeto, execute ```npm i -f```
- Para subir as aplicações, execute ```npm run start:dev```
- Acesse em: [localhost](//localhost:9000/).
